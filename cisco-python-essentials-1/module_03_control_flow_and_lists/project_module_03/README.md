# 🎓 Smart Student Gradebook

## 📌 Overview

This project is my **Module 03 Milestone Project** for the **Cisco Networking Academy – Python Essentials 1** course.

The program simulates a console-based student grade management system. It stores student records using nested lists, calculates academic statistics, assigns grades, and allows searching for student records by name.

The purpose of this project is to apply all major concepts learned throughout **Module 03 (Lists)** by building a practical, real-world Python application.

---

## 🎯 Project Goal

Build a complete Python application using only the concepts covered in **Module 03**, demonstrating the ability to:

* Store and process data using nested lists
* Traverse lists using loops
* Perform calculations on list data
* Search records efficiently
* Generate formatted reports
* Organize a larger Python program

---

## ✨ Features

* Store student names and marks
* Display student records in a formatted table
* Calculate total and average marks
* Assign grades
* Find the class topper and lowest scorer
* Find the highest marks in each subject
* Calculate the class average
* Count passed and failed students
* Search for a student by name (case-insensitive)
