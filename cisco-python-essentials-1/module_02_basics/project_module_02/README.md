# 🧾 Personal Expense Receipt Generator

## 📌 Overview

This project is my **Module 02 Milestone Project** for the **Cisco Networking Academy – Python Essentials 1** course.

The program simulates a simple shopping receipt system by collecting shop details, customer information, and data for three products. It then performs the required calculations and generates a formatted receipt displaying the subtotal, tax, and grand total.

The purpose of this project is to reinforce the fundamental programming concepts learned throughout Module 02 before moving on to more advanced Python topics.

---

## 🎯 Project Goal

Build a complete Python application using only the concepts covered in **Module 02**, demonstrating the ability to:

- Collect user input
- Store data using variables
- Perform arithmetic calculations
- Display formatted output
- Organize a complete program from input to final result

---

## ✨ Features

- Store shop and customer information
- Accept details for three products
- Calculate the total price for each product
- Calculate subtotal
- Apply a fixed 10% tax
- Calculate grand total
- Display a clean shopping receipt
